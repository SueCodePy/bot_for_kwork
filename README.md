Bot for Kwork

Учебный pet-проект: Telegram-бот, который парсит новые кворки с сайта Kwork и присылает уведомления пользователям.
Проект в процессе доработки.

Что умеет бот

собирает новые кворки с Kwork;

отправляет их пользователю в Telegram;

использует FSM для работы с состояниями;

данные хранятся в PostgreSQL через SQLAlchemy;

кнопки и команды для взаимодействия.

Установка и запуск

Клонировать проект:

    git clone https://github.com/SuePyCode/bot_for_kwork.git

    cd bot_for_kwork


Установить зависимости:

    pip install -r requirements.txt


Создать файл .env (пример в .env.example) и указать токен бота, данные для БД и другие переменные.

Запустить:

    python main.py

Используемые технологии

    Python 3.11+

    Aiogram 3

    SQLAlchemy + PostgreSQL

    requests / BeautifulSoup

В планах

фильтрация и поиск по кворкам;

обработка ошибок и логирование;

тесты и рефакторинг;

деплой на сервер.
